# Analysis-on-the-Prevalence-of-Covid-19
COVID-19 Growth &amp; Prevalence Analysis An exploratory data analysis project examining COVID-19 growth patterns, prevalence, and mortality rates across countries using time-series data.

## Goal
The goal of this project is to analyse how COVID-19 spread over time across different countries, identify periods of rapid growth, and compare prevalence and mortality rates on a per-capita basis.

This analysis aims to identify trends that can inform public health decision-making and resource allocation.
